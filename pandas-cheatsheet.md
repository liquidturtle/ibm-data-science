# Data Analysis with Python  
## Cheat Sheet: Importing Data Sets

| Task                                 | Description                                                                                                                                                          | Code Example                                                                                                                                                   |
|--------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Read CSV data set**                | Read a CSV file into a pandas DataFrame                                                                                                                              | ```df = pd.read_csv(<CSV_path>, header=None)  # load without header<br>df = pd.read_csv(<CSV_path>, header=0)    # load using first row as header``` |
| **Note**                             | In JupyterLite you must download the file locally and use the local path. In JupyterLab or a local Python environment, you can use the URL directly as `<CSV_path>`. |                                                                                                                                                                |
| **Print first few entries**          | Print first *n* rows (default 5)                                                                                                                                     | ```df.head(n)  # default n=5```                                                                                                                      |
| **Print last few entries**           | Print last *n* rows (default 5)                                                                                                                                      | ```df.tail(n)  # default n=5```                                                                                                                      |
| **Assign header names**              | Assign header names to the DataFrame                                                                                                                                 | ```df.columns = headers```                                                                                                                           |
| **Replace "?" with NaN**             | Replace `"?"` with `np.nan`                                                                                                                                          | ```df = df.replace("?", np.nan)```                                                                                                                   |
| **Retrieve data types**              | Get the data types of each column                                                                                                                                    | ```df.dtypes```                                                                                                                                      |
| **Retrieve statistical description** | Summary statistics (numeric by default). Use `include="all"` for all columns.                                                                                        | ```df.describe()             # default<br>df.describe(include="all")```                                                                              |
| **Retrieve data set summary**        | Summary information for the DataFrame                                                                                                                                | ```df.info()```                                                                                                                                      |
| **Save DataFrame to CSV**            | Save processed DataFrame to a CSV file                                                                                                                               | ```df.to_csv(<output CSV path>)```                                                                                                                   |
